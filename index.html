I am a full-stack web developer with a passion for analyzing big data. I want to create data-driven visualizations which provide narratives that effectively communicate complex ideas.
In Novemer of 2014, I started a 12-week, full time immersive web-development bootcamp at General Assembly. I developed Ruby on Rails applications using PostgreSQL. I built applications with JavaScript frameworks and libraries, including jQuery, Backbone, Underscore, D3, Leaflet, and React. My applications were able to connect to web APIs from Twitter, OpenStreetMap, Everytrail, FreeGeoIP, and others.
I've always been interested in technology and the evolution of the web. As a former journalist, I wanted to learn a skill that I could use to communicate my ideas directly with the public. It was amazing to be able to translate my product dreams into code, and getting immediate feedback from experienced developers and team members. I loved the intense learning environment that fired all my pistons the right way. I can't wait to use my skills to make apps that I can use to inform the public with better data visualizations and analysis.
